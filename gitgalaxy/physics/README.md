This directory contains the mathematical core of the **blAST Engine**. 

Once the `/core` optical hub has sliced the source code into text signals, this engine takes over. It applies the GitGalaxy physics equations to generate 0-100% risk exposures, classifies architectural intent using Machine Learning, and maps the temporal churn of the repository over time.

> **⚠️ Configuration Warning:** Do not modify these core physics files to tune the engine's behavior. Almost all variables, Sigmoid curve slopes, risk thresholds, and path modifiers have been abstracted to the **[Standards Registry](../standards/README.md)**. 

### 🗺️ The Architecture

* **`signal_processor.py`:** The central nervous system. It takes the raw regex hits (e.g., number of branches, number of allocations) and applies Sigmoid curves and domain-specific path modifiers to calculate Cognitive Load, Tech Debt, and Security risks. It also executes the bi-axial anomaly detection to find "Alien" files hiding in foreign ecosystems.
  * 📖 **[Read the Signal Processing Equations](https://squid-protocol.github.io/gitgalaxy/02-09-signal-processing/)**

* **`chronometer.py`:** The temporal engine. It interfaces with the local `.git` history using a high-speed streaming escalator to calculate the pulse rate (churn) and stability of individual files over a rolling dynamic window.
  * 📖 **[Read the Chronometer Mechanics](https://squid-protocol.github.io/gitgalaxy/02-15-chronometer/)**

* **`neural_auditor.py`:** The machine learning inference engine. It uses pre-trained K-means clustering models to assign every file and function to a specific "Archetype" (e.g., *The God Node*, *Declarative Glue*, *Async Orchestrator*). It also contains sensors to surgically audit massive LLM weights (`.safetensors`, `.gguf`) without loading them into RAM.
  * 📖 **[Read the Neural Auditor & Archetypes Specs](https://squid-protocol.github.io/gitgalaxy/02-19-neural-auditor/)**

* **`spectral_auditor.py`:** The quality control gate. It uses Bayesian Accountability and the "50/0 Law" to detect Linguistic Drift. If a file acts as a statistical outlier compared to its peers (e.g., a massive data dump disguised as code), it is banished to the Singularity to prevent it from warping the 3D map.
  * 📖 **[Read the Spectral Audit Specs](https://squid-protocol.github.io/gitgalaxy/02-11-spectral-audit/)**
