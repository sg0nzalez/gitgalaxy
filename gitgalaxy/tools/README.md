Welcome to the action layer of the **GitGalaxy Ecosystem**.

If the `core` and `physics` directories are the telescope mapping the stars, this directory contains the automated drones sent out to harvest them. 

GitGalaxy operates on a modular Hub-and-Spoke architecture. The central **blAST Engine** builds a deterministic, mathematical graph of the repository in RAM. These specialized toolsets hook into that graph to execute high-speed, enterprise-grade operations—solving massive problems that rigid ASTs and hallucinating LLMs cannot handle.

### 🛠️ The Enterprise Toolsets

Click into any directory below for specific Quickstart guides, architecture breakdowns, and CLI commands.

* **🛡️ [`/ai_guardrails`](ai_guardrails/README.md):** Specialized keyword sensors protecting both your application and your codebase. The AppSec Sensor detects weaponized LLM features (RCE funnels), while the Dev Agent Firewall restricts autonomous coding agents from destroying complex files.
* **🕰️ [`/cobol_to_cobol`](cobol_to_cobol/README.md):** A mathematical x-ray suite for sanitizing mainframe monoliths. It safely neutralizes legacy lexical traps, extracts dead "Graveyard" memory, and maps topological DAG execution orders.
* **🚀 [`/cobol_to_java`](cobol_to_java/README.md):** A deterministic, high-fidelity translation pipeline. It converts legacy COBOL into fully compiling, modern Spring Boot architectures, mapping memory exactly before utilizing AI to translate isolated business logic.
* **📜 [`/compliance`](compliance/README.md):** A Zero-Trust Software Bill of Materials (SBOM) generator. It refuses to blindly trust `package.json`, instead locating the physical dependencies on disk and generating strict CycloneDX 1.4 JSON reports.
* **🕸️ [`/network_auditing`](network_auditing/README.md):** A deterministic mapping tool that hunts undocumented vulnerabilities. It uses structural regex to isolate critical Shadow APIs and outdated Ghost APIs.
* **🧱 [`/supply_chain_security`](supply_chain_security/README.md):** Extreme-velocity pre-commit firewalls. Scans physical internals to block steganography, sub-atomic XOR decryption loops, and exposed cryptographic vaults before they enter CI/CD.
* **📊 [`/terabyte_log_scanning`](terabyte_log_scanning/README.md):** Unindexed, tactical log analysis operating at extreme velocity. It streams massive database dumps to deterministically hunt PII leaks and prove exact runtime execution frequencies.
