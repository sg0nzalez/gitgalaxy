# 🏛️ The Museum of Code

Welcome to the Museum of Code. Software is often judged by its UI or its documentation, but here, we look at the raw physical reality of the code itself. 

Using the **GitGalaxy blAST engine**, we have stripped away the abstractions of history's most important and culturally significant open-source repositories to visualize their structural physics, technical debt, and architectural choke points.

### The Exhibits

* 🧬 **[AlphaFold (2018)](teardown-of-alphafold.md)** - The 5,000-line Python breakthrough that solved protein folding.
* 📱 **[Android](teardown-of-android.md)** - The world's most ubiquitous mobile OS.
* 🚀 **[Apollo 11](teardown-of-apollo-11.md)** - The AGC assembly code that took humanity to the moon.
* 🪙 **[Bitcoin v0.1.0](teardown-of-bitcoin.md)** - The original, highly coupled C++ prototype of decentralized finance.
* 🔬 **[BLAST](teardown-of-blast.md)** - The high-performance C/C++ engine decoding genomics.
* 👹 **[DOOM](teardown-of-doom.md)** - The 1993 C monolith that pioneered real-time 3D rendering.
* 🏦 **[Fineract](teardown-of-fineract.md)** - The sprawling Java core banking platform.
* 😈 **[FreeBSD](teardown-of-freebsd.md)** - The legendary 13.8-million-line OS monolith.
* 🔐 **[GnuPG](teardown-of-gnupg.md)** - The strict C architecture securing the internet's cryptography.
* ☁️ **[Kubernetes](teardown-of-kubernetes.md)** - The massive Go orchestration engine powering the cloud.
* 🐧 **[Linux](teardown-of-linux.md)** - The 24-million-line C titan that runs the world.
* 🌐 **[MediaWiki](teardown-of-mediawiki.md)** - The PHP architecture behind Wikipedia.
* 👁️ **[NVDA](teardown-of-nvda.md)** - The Python/C++ hybrid making Windows accessible.
* 👁️‍🗨️ **[OpenCV](teardown-of-opencv.md)** - The computer vision library.
* 🐼 **[Pandas](teardown-of-pandas.md)** - The God Nodes of the Python data science ecosystem.
* 🛤️ **[Ruby on Rails](teardown-of-rails.md)** - The web framework that championed convention over configuration.
* ⚛️ **[ROOT](teardown-of-root.md)** - CERN's C++ data analysis framework.
* 🗄️ **[SQLite](teardown-of-sqlite.md)** - The most deployed database engine in human history.
* 🧠 **[TensorFlow](teardown-of-tensorflow.md)** - The heavily coupled AI operations monolith.
* 📝 **[WordPress](teardown-of-wordpress.md)** - The PHP engine running a third of the web.
* 🌪️ **[WRF-Fortran](teardown-of-wrf-fortran.md)** - The weather forecasting model.
