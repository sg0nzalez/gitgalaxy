# Real-World Output: IBM CICS Banking Translation

This directory contains the unedited, raw output generated by the GitGalaxy Mainframe Modernization Suite against a standard IBM CICS COBOL application.

* **`/phase_1_clean_room/`**: The output of the `cobol_refractor_controller.py`. Contains the extracted PostgreSQL schemas, zero-trust JCLs, and isolated business logic JSON slices.
* **`/phase_2_spring_boot/`**: The output of the `cobol_to_java_controller.py`. Contains the fully scaffolded, Maven-compilable Spring Boot architecture mapped directly from the Phase 1 artifacts.

Feel free to browse the generated `src/main/java` directory to inspect the automatic JPA Entity mapping and REST Controller scaffolding.
