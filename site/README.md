# GitGalaxy: WebGPU Visualizer (GitGalaxy.io)

This directory contains the source code for the **GitGalaxy 3D Visualizer**, a high-performance WebGPU/WebGL rendering engine designed to map repository telemetry in real-time. 

A live, hosted version of this visualizer is available at **[GitGalaxy.io](https://gitgalaxy.io/)**.

### Zero-Trust & Local Execution
This visualizer is strictly client-side. When you drop a `your_repo_GPU_galaxy.json` file into the interface, **zero data is transmitted**. All parsing, rendering, and spatial calculations happen locally in your browser's memory using `js/core/galaxy-engine.js` and `phase-6-shaders.js`.

### Running Locally (Development)
If you want to run the visualizer locally or modify the UI/shaders:

1. Navigate to this directory.
2. Boot the local server (e.g., using the included `app.py` or a standard Python HTTP server).
~~~bash
python3 app.py
~~~
3. Open `localhost` in your browser.
4. Drag and drop any JSON state dump generated by the `galaxyscope` CLI.
